# By dimension

Generic BASE filters on `dimensions:` (no industry jargon).

| Dimension | Values (vocab) | Pilot use |
|-----------|----------------|-----------|
| **Business** | crud · workflow · permission · business-rule | workflow on login TCs |
| **Technical** | validation · ui · api · … | ui + validation |
| **Quality** | security · performance · reliability · … | security on bad-password |

Deep API / service suites stay in **BE code**, not this hub.
